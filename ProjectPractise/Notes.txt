Types of constructor 
1. System Define: this type of constructor will be created by class itself.

class A
{

}
.... after compilation

class A
{
A()
 {
 }
}

2. User Define: This type of constructor block define by user to initialize dynamic data member of class.

Type of user define costructor
1. static: this type of constructor will be called w/out object and used to initialize static data member of class.

class A
{
	static A()
	{
	//CODE BLOCK
	}
}

2. non static: this type of constructor will be called with object and used to initialize dynamic data member of class.

class B 
{
	B() //non static default constructor
	{
	}
}
2. Parameterized constructor:
class B 
{
	B(int x, int y)
	{
	//code
	}
}
B obj = new B(10, 20);

3. Copy Constructor: it is used to copy the reference of one object to another object.
class A
{
	int a,b;
	A()
	{
	a=10;
	b=20;
	}

A(A o) //copy constructor
{
this.a = o.a;
this.b = o.b;
}
}

class program
{
	psvm()
	{
		A obj = new A();
		A obj1 = new A(obj); //obj1 = obj shallow copy.
	}
}
