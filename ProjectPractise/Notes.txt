4. Copy Constructor: it is used to copy the reference of one object to another object.
class A
{
	int a,b;
	A()
	{
	a=10;
	b=20;
	}

A(A o) //copy constructor
{
this.a = o.a;
this.b = o.b;
}
}

class program
{
	psvm()
	{
		A obj = new A();
		A obj1 = new A(obj); //obj1 = obj shallow copy.
	}

Data Encapsulation: to hide internal data member of class and access it using method or properties.

we will create private data member to implement data encapsulation.

class A
{
	private int a;
}

a is the private data member it can be accessed using method or property. "a" data member cannot access directly.

class A
{

public int A{

Polymorphism: it is the ability of an object to take many forms. 
It allows us to perform a single action in different ways.

TYPES:

1.Static polymorphism: Function overloading is the example of static polymorphism using this we will create same name method with different parameters.
Constructor is the best example of function overloading bcoz constructor name is same as class name but parameters are different.

class Addition
{
	void Add(int a, int b)
	{
	Console.WriteLine(a+b);
	}
	void Add(int a , int b , int c)
	{
	Console.WriteLine(a+b+c);
	}
	void Add(float a , float b , int c)
	{
	Console.WriteLine(a+b+c);
	}
}

2. Dynamic Polymorphism: Functions overriding and overhiding is the example of dynamic polymorphism using this we will create same name method 
with same parameters in parent and child class.

class OldPhone
{
	virtual void fun()
	{
	//old code
	}
}

class NewPhone : OldPhone
{
	override void fun() //function overriding
	{
	//new code
	}
}

Inheritance: it is the process of creating new class from existing class. 
used to reduce code reusability and to achieve data abstraction and polymorphism.

Types:
1. Single Inheritance: when a child class inherits from single parent class.
2. Multilevel Inheritance: when a child class inherits from parent class and that parent class is also child of another parent class.
3. Hierarchical Inheritance: when multiple child class inherits from single parent class.

Assignment:
Inherit Admin(id, name) , Manager(id, name, salary) and employee(Id, name, salary, incentive).
